# Manim UCP Arvin - 用户使用手册

本手册提供各类数学函数输入的示例和参数配置建议。

================================================================================
一、输入类型说明
================================================================================

【显函数输入】
格式: y = f(x)
变量: x
示例: x**2, sin(x), exp(x)

【隐函数输入】
格式: F(x, y) = 0（输入左边部分）
变量: x, y
示例: x**2 + y**2 - 4（表示圆 x² + y² = 4）

【极坐标输入】
格式: r = r(θ)
变量: theta（弧度制）
示例: 2 + 2*sin(theta)（心形线）

================================================================================
十一、常见问题解答
================================================================================

Q1: 为什么我的隐函数画不出来？
A1: 可能原因：
    - 表达式格式错误，确保使用 ** 表示幂运算
    - 坐标范围设置不当，尝试扩大范围
    - 曲线在给定范围内不存在

Q2: 极坐标图形不完整怎么办？
A2: 尝试调整 θ 范围：
    - 标准图形: 0 到 6.28 (2π)
    - 多圈螺线: 0 到 12.56 或更大

Q3: 三维曲面渲染很慢？
A3: 建议：
    - 降低采样点数
    - 缩小坐标范围
    - 使用较简单的函数

Q4: 支持哪些数学函数？
A4: 支持的函数列表：
    - 三角函数: sin, cos, tan
    - 反三角函数: arcsin, arccos, arctan
    - 指数对数: exp, log
    - 其他: sqrt, abs
    - 常量: pi, e

七、三维黎曼积分可视化
================================================================================

【功能说明】
三维黎曼积分可视化用于展示曲面 z = f(x, y) 下方体积的近似计算过程。
基于 Wave of Boxes 风格，实现动态递进的 box 效果展示。

【实现特点】
1. Wave of Boxes 风格动画效果
2. 分层递进展示：从较大 box 过渡到更小更细长的 box
3. 尺寸递进：2.0 → 1.0 → 0.5（单位）
4. 密度递增：box 数量逐渐增加，呈现更高密度
5. 颜色渐变：基于高度的蓝-绿-黄-红颜色渐变
6. 平滑过渡：串行淡出淡入动画实现层级间转换

⚠️ 【已知说明】
- 高密度阶段（0.5尺寸）立方体数量多，边缘可能存在极轻微的视觉接触
- 变换过程采用间隙窗口策略，已最大程度减少重叠
- 推荐使用坐标范围: -6 到 6（更小范围获得更好效果）
- 如遇视觉问题，请尝试缩小坐标范围参数

【动画参数】
--------------------------------------------------------------------------------
参数              | 默认值 | 说明
--------------------------------------------------------------------------------
坐标范围 X,Y     | -6~6  | box 分布范围（更小范围获得更好效果）
动画时长         | 12秒  | 总动画时长
相机 φ           | 56°   | 俯仰角（与z轴夹角）
相机 θ           | -50°  | 方位角（绕z轴旋转）
背景色           | 深灰色 | DARK_GRAY
--------------------------------------------------------------------------------

【阶段说明】
1. 第一阶段：box 尺寸 2.0，间隙 0.4
   - 较大的立方体，稀疏分布
   - 数量少，便于观察整体结构

2. 第二阶段：box 尺寸 1.0，间隙 0.2
   - 中等尺寸，中等密度
   - 开始展现细节

3. 第三阶段：box 尺寸 0.5，间隙 0.1
   - 小尺寸，高密度
   - 更细长的视觉效果
   - 更精确的体积近似

【相机设置】
- 距离: 50（更远的观察距离，便于全景观察）
- 视角: 56°俯仰角 + -50°方位角
- 效果: 类似 Wave_of_boxes_02 的经典观察角度

【视觉效果】
- 背景: 深灰色 (DARK_GRAY)
- 边框: 白色细线 (width=0.5)
- 填充: 80% 透明度
- 颜色渐变: BLUE → GREEN_D → YELLOW → RED

【使用示例】
--------------------------------------------------------------------------------
场景：三维场景
输入方式：显函数
函数表达式：sin(sqrt(x**2 + y**2)) / (sqrt(x**2 + y**2) + 0.5)
功能：积分展示
X轴范围：-6 到 6
Y轴范围：-6 到 6
--------------------------------------------------------------------------------
适用函数推荐：
1. sinc 类函数：sin(R)/R 类型产生漂亮的波动效果
2. 二次函数：x**2 + y**2 产生平滑递增的高度
3. 波动函数：sin(x)*cos(y) 产生网格状高度分布
--------------------------------------------------------------------------------

【功能说明】
在三维场景下，支持所有输入类型（显函数、隐函数、极坐标、参数方程）使用曲面变换功能。
勾选 "曲面变换" 后，显示平面2输入框，输入第二个曲面函数即可实现平滑变形。
使用 Manim 的 ReplacementTransform 技术实现高质量的变形过渡效果。

【实现原理】
1. 两个曲面使用相同的 u, v 参数化
2. 保证拓扑结构完全一致（相同的 resolution）
3. Manim 自动计算顶点间的映射关系
4. ReplacementTransform 实现顶点位置的平滑插值

【支持的输入类型】
* 显函数模式：平面1和平面2均输入 z = f(x, y) 形式
* 隐函数模式：平面1输入隐函数 F(x,y,z) = 0，平面2输入 z = f(x, y) 形式
* 极坐标模式：平面1输入极坐标表达式，平面2输入 z = f(x, y) 形式
* 参数方程模式：平面1输入参数方程，平面2输入 z = f(x, y) 形式

【使用步骤】
1. 切换到 "三维场景" （注意：三维场景自动隐藏泰勒展开功能）
2. 选择任意输入类型：显函数 / 隐函数 / 极坐标 / 参数方程
3. 在功能选项中勾选 "曲面变换"
4. 此时会显示 "函数表达式 - 平面2" 输入框
5. 在 "平面1" 输入区域输入第一个曲面（与选择的输入类型对应）
6. 在 "平面2" 输入框输入第二个曲面函数（统一使用 z = f(x, y) 显函数形式）
7. 点击 "开始生成动画"

【示例】

示例 1: 抛物面 → 双曲面
  平面1: x**2 / 4 - y**2 / 4      # 抛物面
  平面2: x**2 / 4 + y**2 / 4      # 双曲面
  X范围: -6,6
  Y范围: -6,6
  Z范围: -10,10

示例 2: 正弦波 → 余弦波
  平面1: sin(x) * cos(y)         # 正弦曲面
  平面2: cos(x) * sin(y)         # 余弦曲面
  X范围: -6,6
  Y范围: -6,6
  Z范围: -2,2

示例 3: 平面 → 马鞍面
  平面1: 0                       # 水平面
  平面2: x*y / 4                 # 马鞍面
  X范围: -5,5
  Y范围: -5,5
  Z范围: -5,5

示例 4: 球面 → 抛物面
  平面1: sqrt(16 - x**2 - y**2)  # 上半球
  平面2: (x**2 + y**2) / 4       # 抛物面
  X范围: -4,4
  Y范围: -4,4
  Z范围: 0,5

【效果特点】
- 平滑变形: 4秒过渡时间，视觉效果流畅
- 颜色区分: 平面1使用黄色(YELLOW_D)，平面2使用蓝色(BLUE)
- 旋转展示: 变形结束后自动旋转视角展示效果
- 拓扑一致: 相同参数化确保变形自然

【注意事项】
1. 两个函数的定义域应尽可能匹配，避免边界突变
2. 推荐使用相同或相似的坐标范围
3. 变形效果取决于两个曲面的拓扑相似性
4. Z范围应能容纳两个曲面的极值
5. 避免奇点和不连续点，确保函数在整个定义域内可计算
