====================用户操作相关信息======================

零、环境配置相关信息
    本程序基于python及其第三方库开发，主要包括为Manim、等等，
    此程序已经上传github，可在github上进行

一、主要相关程序
    用户主要使用GUI界面，目前主要是通过激活GUI_new.py文件
    （后期将变更为GUI_pro.py）来进行相关控制。用户仅面向此区域
    进行数据的相关输入。

    目前用户使用的是GUI_new.py文件，此GUI界面文件为引导激活cmd
    来进行相关参数输入。目前此界面的对应按钮的使用信息如下：
    
    |二维显式函数|二维隐函数|极坐标参数方程|
    |二维显式积分|极坐标积分| 三维图形绘制 |
    |  显式求导  |隐函数求导| 三维相机设置 |

    下面将对于每个按钮提供的相关参数进行具体讲解相关参数机器效果。

二、使用方式
    （1）二维显式函数
        二维显式函数需要传入总计七个参数，需要遵循默认的相关参数
        来进行设置。目前的自变量为x，因变量为y。

        需要传入的七个参数如下

        x轴左边界（默认为-10.0）
        x轴右边界（默认为10.0）
        x轴步长（默认为1.0）

        y轴左边界（默认为-10.0）
        y轴右边界（默认为10.0）
        y轴步长（默认为1.0）

        显式函数 y = f(x) (例如 x ** 2)

        传入这几个参数之后，就会自动进行相关场景的绘制。

    （2） 二维隐函数
        二维隐函数需要传入七个参数如下：

        x轴左边界（默认为-10.0）
        x轴右边界（默认为10.0）
        x轴步长（默认为1.0）

        y轴左边界（默认为-10.0）
        y轴右边界（默认为10.0）
        y轴步长（默认为1.0）

        隐函数方程（使用x和y变量）
        注意，在进行隐函数方程输入时，只需要输入隐函数的非零部分
        如，在输入 x ** 3 + y ** 3 = 6*x*y 时，需要输入的相关
        信息是
                x ** 3 + y ** 3 - 6*x*y 
        
        在完成相关参数传入后，便可以完成相关场景的绘制。

    （3） 极坐标参数方程
        需要用户输入半径最大值、角度步长、半径步长和参数方程。
        输入方程需要使用自变量theta，这个是作为自变量的角度。
        下面的一些输入实例

        四叶玫瑰线：2*cos(2*theta)
        双纽线：sqrt(4*cos(2*theta))
        圆锥曲线：1/(1 + 0.8*cos(theta))
        复杂函数：exp(-theta/5)*cos(3*theta)

        目前支持的相关运算除了正常的四则运算和乘方外，还可以使用以下
        运算：

        sin, cos, exp, sqrt, log

        这几种运算相关用法也可以通过Latex了解相关含义。
        在完成相关参数传入后，便可以完成相关场景的绘制。

    （4） 二维显式积分
        和前面二维显式函数类似，没有区别。

    （5） 极坐标积分
        和前面极坐标参数方程类似，没有区别。

    （6） 三维图形绘制
        这是可以理解为参数表啥，建议使用默认的u、v来进行相关绘制。

        我们给出输入格式：
        === 三维曲面参数输入 ===
        （表达式可以使用sin/cos/exp/sqrt/log等函数）
        （示例：x=u, y=v, z=u**2 + v**2）

        请输入u参数名称（默认：u）: 
        请输入v参数名称（默认：v）: 
        输入x的表达式（使用u/v变量）: 
        输入y的表达式（使用u/v变量）: 
        输入z的表达式（使用u/v变量）: 
        输入u最小值（默认：-2）: 
        输入u最大值（默认：2）: 
        输入v最小值（默认：-2）: 
        输入v最大值（默认：2）:

        下面给出相关示例

        === 三维曲面参数输入 ===
        请输入u参数名称（默认：u）: [直接回车]
        请输入v参数名称（默认：v）: [直接回车]
        输入x的表达式（使用u/v变量）: (1 + 0.5*v*cos(u/2)) * cos(u)
        输入y的表达式（使用u/v变量）: (1 + 0.5*v*cos(u/2)) * sin(u)
        输入z的表达式（使用u/v变量）: 0.5*v*sin(u/2)
        输入u最小值（默认：-2）: 0
        输入u最大值（默认：2）: 6.28
        输入v最小值（默认：-2）: -1
        输入v最大值（默认：2）: 1

        使用参数方程的原因是为了更好的将绘制，使用原本的x，y，z太过于复杂，
        不方便供程序进行解析。

        利用这个程序，可以将绘制如莫比乌斯环等信息。

    （7） 显式求导
        这个是给二维平面使用。主要是展示导数的几何意义。使用方式和二维平
        面绘制相同。

    （8） 隐函数求导
        目前这个也是二维平面使用，但是图形绘制的时候会出现bug，目前不建议
        使用此功能（因为绘制出来的图形是错误的，而切线是正确的）。

    （9） 三维相机设置
        这个功能后面将会集成到所有三维场景的设置当中。内部设置了一个莫比乌
        斯带，以供用户学习这个设置如何在三维场景中进行使用，主要包括初始时
        的相关设置和中间改变运动状态的设置。
